父母早亡、与祖母相依为命、家庭主要劳动力的缺失造成生活贫困……统计显示：中国57万孤儿，三分之一仍亟待有效救助。在并不富裕的农村，孤儿的到来使寄养家庭经济负担骤然加重，造成新的贫困。而与普通孩子比，农村孤儿入学年龄普遍偏大或多次辍学，他们所在年级与年龄极不相称。 农村孤儿一直是爱德基金会重点关注的弱势群体，2002年开始，爱德基金会启动农村孤儿助养项目。2008年，爱德基金会与腾讯公益携手推出“网助成长”公益月捐，截止2014年2月，已经有超过71万腾讯网友参与捐助。 为保护爱心网友利益，让更多的孤儿得到救助，“贫困孤儿助养”项目承诺： 1、我们确保您的每笔捐赠款项将都交给所资助的孩子 2、我们确保您将可以随时看到所资助孩子的款项接收记录；“贫困孤儿助养”2011下半年款项领取记录为了让这些孩子不仅在学习生活上顺利，也能在心灵成长方面得到照顾，我们还定期为孩子组织“e万行动-天使心灵成长计划”，帮助他们的身心都能快乐发展。
