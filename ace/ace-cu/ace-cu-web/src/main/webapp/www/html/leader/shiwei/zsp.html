朱水平同志的简历

男，汉族，湖南湘阴人，1963年5月出生，1985年3月加入中国共产党，1981年9月参加工作，中南工业大学社会科学系自然辨证法专业毕业，研究生学历，哲学硕士。

1981.09--1985.07 南京空军气象学院学习；

1985.07--1987.09 南京空军气象学院政教室助教；

1987.09--1989.12 中南工业大学社会科学系学习；

1989.12--1990.09 南京空军气象学院政教室教员；

1990.09--1993.07 国防科大政治部宣传处干事；

1993.07--1994.06 国防科大政治部副营职干事；

1994.06--1995.04 湖南省人民政府办公厅人事处副科级干部；

1995.04--1996.12 湖南省人民政府办公厅主任科员；

1996.12--1998.05 湖南省人民政府办公厅秘书二处主任科员；

1998.05--2001.08 湖南省人民政府办公厅秘书二处副处级秘书；

2001.08--2004.04 湖南省人民政府办公厅秘书二处正处级秘书；

2004.04--2008.05 湖南省人民政府驻外办事处联络办公室主任；

2008.05--2008.12 湖南省商务厅党组成员、纪检组长；

2008.12--2010.01 张家界市委常委、组织部部长；

2010.01--2013.07 张家界市委常委、组织部部长，市委党校（市行政学院）第一校（院）长；

2013.07--2014.08 常德市委常委、组织部长；

2014.08--2017.09 常德市委常委、市政府副市长（常务）、党组副书记；
